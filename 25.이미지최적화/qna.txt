Q. 이미지 최적화에 대해 설명하고 방법에 대해 설명하라

[이미지 최적화란?]

이미지 최적화는 웹 사이트의 속도와 성능을 향상시키기 위해 이미지 파일 크기를 줄이는 동시에 이미지 품질을 유지하는 과정을 말함. 
이로써, 사용자는 더 빠른 로딩 시간을 경험하게 되고, 웹 서버는 더 적은 데이터를 전송하게 됨.

[이미지 최적화 진행 과정]

1. 이미지 분석: 사용 중인 이미지의 크기, 포맷, 품질 등을 분석함.
2. 적절한 포맷 선택: 웹에서 사용될 이미지에 가장 적합한 포맷(JPG, PNG, WebP 등)을 선택함.
3. 품질 조절: 높은 품질의 이미지는 파일 크기가 크므로, 웹에 적합한 품질로 조절함.
4. 이미지 리사이징: 필요에 따라 이미지의 해상도를 조절함.
5. 압축: 이미지 파일의 크기를 줄이기 위해 특정 알고리즘을 사용하여 압축함.

[이미지 최적화 방법]

- 파일 형식 변경: 예를 들어, 배경 투명이 필요하지 않은 이미지는 JPG로, 투명이 필요한 이미지는 PNG로 저장함.
- 레이지 로딩 적용: 스크롤을 내릴 때까지 이미지를 로드하지 않는 기술임.
- CDN 사용: 사용자에게 더 빠르게 이미지를 제공하기 위해 Content Delivery Network를 활용함.
- 스프라이트 이미지 활용: 여러 개의 이미지를 하나의 이미지로 합쳐서 네트워크 요청 횟수를 줄임.
- 이미지 최적화 도구 사용: TinyPNG, ImageOptim, Squoosh 등의 도구를 활용하여 이미지 최적화 진행함.
