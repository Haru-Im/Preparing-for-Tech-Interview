
[CDN (Content Delivery Network)이란?]

CDN은 전 세계에 분산된 서버 네트워크로, 사용자가 웹 콘텐츠를 더 빠르게, 안정적으로 접근할 수 있게 도와주는 시스템임. 
사용자가 특정 웹 콘텐츠(예: 이미지, 비디오, CSS, JavaScript 파일 등)에 접근할 때, 가장 가까운 위치에 있는 서버에서 해당 콘텐츠를 제공함. 
이로써 로딩 시간이 단축되고, 웹사이트의 전체적인 성능과 안정성이 향상됨.

[CDN의 예시]

Akamai: 세계 최대 규모의 CDN 서비스 제공자로, 수많은 기업과 웹사이트가 Akamai의 서비스를 활용하고 있음.

Cloudflare: 웹 사이트 보호 및 가속화를 목표로 하는 서비스로, DDoS 방어와 함께 CDN 서비스를 제공함.

Amazon CloudFront: Amazon Web Services (AWS)의 CDN 서비스로, AWS의 다양한 서비스와 통합하여 사용할 수 있음.

Fastly: 실시간 콘텐츠 전송에 특화된 CDN 서비스 제공자로, 많은 인터넷 기업들이 이 서비스를 활용함.

즉, 만약 한국에 있는 사용자가 미국 기반의 웹사이트를 방문한다면, 
CDN이 활성화되어 있다면 해당 사용자에게는 한국 내의 CDN 서버로부터 콘텐츠가 전송될 것임. 
이렇게 되면 미국 서버까지의 거리보다 훨씬 짧아진 거리에서 데이터를 받아오게 되므로 로딩 속도가 향상됨.
